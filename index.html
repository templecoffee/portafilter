Data Merge SOP

  1. Open PF_WholesaleInfoSheet.indd
  2. Window → Utilities → Data Merge
  3. Click the hamburger menu → Select Data Source
  4. Select your converted CSV
  5. Hamburger menu → Create Merged Document
  6. Adjust any errors
  7. Add photos
  8. Add or edit the Farm Story if not added or needs to be shorter
